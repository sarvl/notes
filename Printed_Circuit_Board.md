## Design
 - Always aim at SMD for cheap manufacture.
 - Stick to common components
 - Get reels or tubes
 - Ensure you are within manufacture capabilities
 - Figure out a way to program board once everything is soldered
 - Use spreadsheet
 - Panelize with single design
 - Add enough support so boards don't bend
 - Add fiducials, especially for high density devices like BGA
 - High density pins should use gold because it can be made really flat
 - Pull back copper from the edges
 - Watch out for number of different components
 - Add connectors for testing
