#electronics 

Electronics is a discipline of applying [[Physics]] to manipulate [[Electron|Electrons]], specifically via [[Semiconductor_Device|Semiconductor Devices]] - [[Diode]] and [[Transistor]].

Electronics can be divided into [[Digital_Electronics|Digital]] and [[Analog_Electronics|Analog]] if it the allowed [[Voltage]] is quantized or not, respectively.

## Device Types
1. Passive - do not add [[Power]] to a [[Signal]], eg
	- [[Resistor]]
	- [[Capacitor]]
	- [[Inductor]]
	- [[Diode]]
2. Active - add [[Power]] to a [[Signal]], eg 
	- [[Transistor]]

## Description
Electronic devices can be described with
- [[Equation]] eg $I = V/R$ ([[Ohms_Law]])
- [[Graphical]] 
	- I-V; V-I; V-V; I-I - general characteristic
	- [[Time]] domain - a response to given stimuli
	- [[Frequency]] domain - parameter change as input frequency changes
- Catalog - a set of parameters
