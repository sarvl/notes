#computer_architecture 

Microprocessor without Interlocked Pipeline Stages is an ISA based on typical RISC principles. It is generally simple ISA designed with [[Pipelining|Pipelined]] implementation in mind, hence the name.
MIPS is a simple, load-store architecture.

## Interesting Features
- [[Binary#Overflow|Overflow]] detection built into instructions.
  Arithmetic instructions have variant [[Trap|Trapping]] on overflow.
