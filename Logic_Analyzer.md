#electronics #digital 

Logic analyzer is a tool used primarily to debug [[Digital_Electronics|Digital Circuits]], it differs from [[Oscilloscope]] in number of available channels and precision of stored data.

It is really useful in debugging digital buses, but it does not guarantee the actual correctness of signal.

There are two kinds of analysis:
- Timing Analysis - internal clock, mostly like oscilloscope
- State Analysis - external clock, used to see the system state
