#telecom  

A Virtual [[Computer_Network#Range|LAN]] is a separation created at [[Computer_Network#Reference Models|Link]] layer. It is used to enhance [[Cybersecurity|Security]] and to improve logical grouping of  devices. VLANs can be assigned dynamically - by MAC or by login, or statically - by port.
[[Switch|Switches]] transfer frames only within the same VLAN. With more than one switch in the [[Computer_Network|Network]] it is necessary to either use many cables or a switch that supports trunking.
In a network with many switches, ALL have to have the same set of VLANs created.
To send a data to the same company through ISP, another tag is added to a packet to ensure only the desired company receives it. This tag is used to allow different companies to use the same tags.


## VLAN Hopping
While it should be impossible to move to a different switch, this sometimes is possible during cyberattacks.
