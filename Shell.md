#operating_systems 
Shell is a program responsible for interfacing between user and [[Operating_System|OS]].

Shells may be used 
- interactively - input is typed in by user
- non interactively - input is taken from the file

Shell may execute commands 
- synchronously
- asynchronously

Example shells in [[Linux]]
- sh
- bash
- zsh
- fish

Shells provide small umber of built-in commands that cannot be implemented outside the shell.
