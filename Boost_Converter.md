## Choosing
The chip, has to be able to handle larger [[Current]] and lower [[Voltage]] than the output.

### Criteria
- Efficiency vs Load Current
- Minimum Input Voltage
- Minimum Startup Voltage
- Cost and Availability
- Component Count
- Shutdown Current
- Footprint Size 
- Frequency 
- Power Output
- Noise
- Reference Voltage Temperature Stability
- Transient Response
